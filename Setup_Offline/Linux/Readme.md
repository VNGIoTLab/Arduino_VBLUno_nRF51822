### Cài đặt
- Trong quá trình cài đặt cần chắc chắn phần mềm Arduino IDE không được thực thi.
`cd Setup_Offline_VBLUno_Linux_xxx`
`chmod 777 Setup_VBLUno_Linux`
`sudo ./Setup_VBLUno_Linux`
